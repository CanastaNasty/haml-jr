Tempest - Alpha

Create simple interactive HTML components using CoffeeScript , HamlJr , and Styl

There are still many bugs, but I think this is interesting enough to put out into the world for feedback .

In the data section you return an object that has observable properties. In the template these properties are bound to attributes or text. As your model is updated the rendered template will always stay up to date. The binding is automatic by introspecting the observable properties.
